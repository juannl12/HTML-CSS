O que é CSS inline?

CSS inline é uma maneira de aplicar estilos diretamente a um elemento HTML, utilizando o atributo "style". Isso significa que você adiciona o estilo diretamente na tag do elemento, sem a necessidade de criar uma folha de estilos externa ou interna.
